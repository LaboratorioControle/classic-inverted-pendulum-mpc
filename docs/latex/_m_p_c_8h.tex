\doxysubsection{Detailed Description}
Implementation of a Model Predictive Control (\doxylink{class_m_p_c}{MPC}) controller based on Quadratic Programming using the qp\+OASES solver. 

This library allows the use of different \doxylink{class_m_p_c}{MPC} formulations:\+
\begin{DoxyItemize}
\item Classic \doxylink{class_m_p_c}{MPC}
\item \doxylink{class_m_p_c}{MPC} with linear parameterization (trivial)
\item \doxylink{class_m_p_c}{MPC} with exponential parameterization 
\end{DoxyItemize}
