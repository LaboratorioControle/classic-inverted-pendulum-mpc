\doxysubsection{Detailed Description}
Lightweight dynamic matrix class for embedded systems. 

This class implements a simple matrix structure using a contiguous std::\+vector$<$float$>$ storage in row-\/major format. It provides basic linear algebra operations required for control algorithms such as Model Predictive Control (\doxylink{class_m_p_c}{MPC}).

The class is designed to be lightweight and suitable for microcontrollers, avoiding heavy dependencies from full linear algebra libraries.
